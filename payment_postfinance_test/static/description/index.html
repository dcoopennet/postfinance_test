How to configure Postfinance Backoffice?

Follow these instructions

Under "Global security parameter"

Choose SHA-1 as Hash algorithm

Under "Data and origin verification"

Choose an SHA-IN key

Under "Transaction feedback"

Choose an SHA-IN key

Under "Data and origin verification" tab:

Enable feedback by clicking on "I would like to receive transaction feedback parameters on the redirection URLs."

Choose the following feedback parameters (Dynamic e-Commerce parameters):

ACCEPTANCE, AMOUNT, BRAND, CARDNO, CURRENCY, NCERROR, ORDERID, PAYID, PM, STATUS

Choose an SHA-OUT key
